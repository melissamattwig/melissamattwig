Surface and Groundwater Contamination Risk Index in Webster Township, MI
Surface and Groundwater Contamination Risk Index
This risk index was developed using a weighted raster overlay of three variables:
1) soil texture (source: SSURGO), which is used as a proxy for water percolation
2) elevation (DEM from the State of Michigan GIS Open Data Portal), which affects lateral movement of water and
3) nearness to water (source: MIRIS), which determines how much movement is required to reach surface water bodies.

This project involved converting vector data to raster format, joining tables, combining raster grids using Spatial Analyst Tools, and stretching raster grids so each variable is normalized.

All analyses were performed using ESRI's ArcGIS Pro.
Land Use Land Cover Change
LULC Change in Mill Creek Watershed between 2001 and 2011
Land cover data for this project was retrieved from the Natioanl Land Cover Database, reclassified, then converted to vector format. Clips, buffers, and intersections were used to delineate features and trim the dataset to the Mill Creek Watershed. Summary statistics were used determine percent land cover change between the two study periods.

All analyses were performed using ESRI's ArcGIS Pro.
